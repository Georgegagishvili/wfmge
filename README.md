# WFMGE

A web-based marketplace for community items, inspired by warframe.market, built with **Flutter Web** for frontend and **(TBD)** for backend.

---

## Features (TBD)

- Browse and search items with filters
- View active buy/sell orders for each item
- User authentication (JWT / OAuth)
- Create, update, and cancel orders
- User profiles with reputation
- Realtime presence and order updates (WebSockets)
- Price charts for items (MVP-ready)
